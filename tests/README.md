# Tests

Thư mục này dành cho test cấp hệ thống (cross-service / end-to-end) của dự án.

Ghi chú:

- `backend/tests/` dùng cho test riêng của backend package (đang có test thật).
- `tests/` ở root dùng cho test cross-service hoặc end-to-end.
- Khi thêm test, tạo lại các thư mục con (`unit/`, `integration/`, `e2e/`) theo nhu cầu.
