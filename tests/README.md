# Tests

Thư mục này dành cho test cấp hệ thống của dự án.

```text
tests/
├── unit/
├── integration/
└── e2e/
```

Ghi chú:

- `backend/tests/` dùng cho test riêng của backend package.
- `tests/` ở root dùng cho test cross-service hoặc end-to-end.
- Các thư mục hiện là placeholder để giữ cấu trúc cho các phase sau.
